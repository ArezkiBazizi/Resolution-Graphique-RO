# Resolution-Graphique en recherche Operationnelle

- Donnez la taille maximale des abscisses et des ordonnées à afficher.
- Donner le nombre de contraintes.
- Donner les (n) contraintes ( exemple : ax+b<=c).
  
- Exécution de l'algorithle :
- dessine les droites et les points d'intersection.
- efface les zones qui ne sont pas de solutions.
- met en avant les espaces qui sont des solution et identifie les points qui sont potentiellement des solutions solutions optimales.
- comapare les points selon l'objectif et selectionne la solution optimale.
- dessine la solution optimale en bleu.
- affiche la solution optimale.
